# JSON Array-parser

Приложение позволяет пользователю загружать json-файл (массив объектов) и рендерит его в виде таблицы.
Пользователь может просмотреть/отредактировать любые данные в строке таблицы и сохранить как новый файл.
Посмотреть онлайн можно здесь:
https://enmalafeev.github.io/array-parser/

## Запуск проекта

1. Убедитесь, что установлены Node.js 22.11.0.
2. Склонируйте репозиторий.
3. Установите зависимости: `npm install`.
4. Запустите dev-сервер: `npm run dev`.
5. Откройте http://localhost:5173.
